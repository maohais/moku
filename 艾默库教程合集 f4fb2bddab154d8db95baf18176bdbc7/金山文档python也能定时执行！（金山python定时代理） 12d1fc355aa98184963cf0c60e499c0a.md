**简述**

文章中的代码已放入github仓库中

```
https://github.com/imoki/wpsPython
```

金山文档目前的python脚本无法设置为定时任务。

此脚本消除了金山文档的airscript调用python脚本的壁垒，由于目前python不能设置定时，导致许多python开发者转向不熟悉的airscript开发，而本程序能够使开发者直接开发python脚本，并利用airscript的定时任务来运行python脚本。更多使用场景请尽情想象。

**从而达到python也能定时执行的效果**！使用非常简单。
